# My Portfolio

A personal portfolio website built with React, TypeScript, and Vite.

## Features

* Responsive design
* About Me section
* Skills showcase
* Projects portfolio
* Contact form with Web3Forms

## Tech Stack

* React
* TypeScript
* Vite
* Web3Forms

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
